# Modal

The Modal component provides a main Modal wrapping element for layout and
styling modal content. It is composed of anchor-ui Box and Header elements
as well as the Modal element from ReactModal. Additionally, a `useModal` hook is
available to provide state control for the modal, giving a user to access the
`closeModal` and `openModal` functions.

## Import

```javascript
import { Modal, useModal } from "@buoysoftware/anchor-ui-modal";
```
