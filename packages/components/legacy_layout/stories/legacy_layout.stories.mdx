# LegacyLayout

The LegacyLayout component provides a compatability bridge to migrate existing
applications to use AnchorUI incrementally. It maps the display variants used in
legacy components to the new AnchorUI equivalents.

## Import

```javascript
import { LegacyLayout } from "@buoysoftware/anchor-ui-compat";
```
