# LegacyButton

The LegacyButton component provides a compatability bridge to migrate existing
applications to use AnchorUI incrementally. It maps the size and color scheme
variants used in legacy components to the new AnchorUI equivalents.

## Import

```javascript
import { LegacyButton } from "@buoysoftware/anchor-ui-compat";
```
