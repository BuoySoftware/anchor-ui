# LegacyText

The LegacyText component provides a compatability bridge to migrate existing
applications to use AnchorUI incrementally. All of the legacy text variants in
Buoy are mapped to a new typography component.

## Import

```javascript
import { Button } from "@buoysoftware/anchor-ui-compat";
```
